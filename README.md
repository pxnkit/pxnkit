# Hi, I'm Pankit Brahmkhatri

I am pursuing an MSc in Computer Science at TU Dresden. I also work as a student research assistant at the Chair of Materials Science and Nanotechnology.

I live in Dresden, Germany, and use he/him pronouns.

I am interested in AI agents that can find useful information, remember what matters, learn from feedback, and make better decisions. My current interests are adaptive evidence seeking, agent memory, test-time learning, and search-guided reasoning.

## What I am exploring

- When should an agent search, and how should it choose a source?
- What should an agent remember, update, or forget?
- How can feedback from one task improve later decisions?
- How can retrieval and verification make reasoning more dependable?

## Tools and technologies

These are the tools and methods I use or am learning for this work.

- **Programming and engineering:** Python, SQL, Git, Linux, FastAPI, Pydantic, pytest, Ruff, mypy, Docker, GitHub Actions, and AWS

- **Evaluation and safety:** BEIR, ir_datasets, ir_measures, pytrec_eval, ranx, MLflow, OpenTelemetry, Recall@k, MRR, nDCG, calibration, sandboxing, prompt injection testing, and memory poisoning testing
